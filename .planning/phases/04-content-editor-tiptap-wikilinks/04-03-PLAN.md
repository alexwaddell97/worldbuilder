| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-08 | XSS | TiptapEditor renders content via EditorContent | mitigate | Tiptap renders through React DOM and ProseMirror — not innerHTML. Wikilink labels via JSX {node.attrs.label} — React escapes. |
| T-04-09 | XSS | WikilinkAutocomplete renders item.name | mitigate | Rendered as JSX text children — React escapes all strings; no innerHTML |
| T-04-10 | DoS | Auto-save onUpdate fires on every keystroke | mitigate | 2-second debounce via setTimeout/clearTimeout — prevents flooding saveEntityContentAction |
| T-04-11 | IDOR | saveEntityContentAction called by TiptapEditor | mitigate | Server Action verifies worldId ownership before update (Plan 01); client cannot bypass this |

| T-04-SC | Tampering | No new npm installs | accept | All packages installed in Plan 01 |
